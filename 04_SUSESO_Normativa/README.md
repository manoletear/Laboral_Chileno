# 04 - Normativa SUSESO (Superintendencia de Seguridad Social)

**Sitio oficial:** https://www.suseso.cl/601/w3-channel.html
**Normativa:** https://www.suseso.cl/612/w3-channel.html

## Rol de SUSESO

Fiscaliza a mutualidades (ACHS, Mutual, IST), ISL, CCAF y administra el Compendio de Normas del Seguro de la Ley 16.744. Tambien regula las LME (Licencias Medicas Electronicas).

## Compendio de Normas del Seguro Ley 16.744

El Compendio sistematiza todas las instrucciones sobre prestaciones preventivas, economicas y medicas.

| Libro | Materia |
|-------|---------|
| Libro I | Disposiciones generales |
| Libro II | Normas sobre mutualidades e ISL |
| Libro III | Denuncia, calificacion y evaluacion de incapacidades permanentes |
| Libro IV | Prestaciones preventivas (gestion riesgos, capacitaciones) |
| Libro V | Prestaciones para trabajadores independientes |
| Libro VI | Prestaciones economicas (subsidios, indemnizaciones, pensiones) |
| Libro VII | Normas sobre financiamiento |
| Libro VIII | Prestaciones medicas |
| Libro IX | Sistemas de informacion, informes y reportes (SISESAT) |
